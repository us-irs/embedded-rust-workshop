Embedded Rust Workshop
==========

This repository contains all components of the Embedded Rust Workshop.

This includes:

- [`slides`](./slides) contains introduction slides about this workshop and embedded Rust.
- [`exercise-book`](./exercise-book) contains the accompanying exercise book.
- [`microbit-code`](./microbit-code/) contains all example source code and exercises which run on
  the microbit v2 target hardware
- [`host-client`](./host-client/) contains a host client used to communicate with the microbit v2 via a serial
  interface.
