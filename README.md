Embedded Rust Workshop
==========

This repository contains all components of the Embedded Rust Workshop.

This includes:

- [`microbit-exercises`](./microbit-exercises/) contains all example source code and exercises which
  run on the microbit v2 target hardware
- [`microbit-apps`](./microbit-apps/) contains sample apps which you can also use to test the board
  setup.
- [`exercise-book`](./exercise-book) contains the accompanying exercise book.
- [`slides`](./slides) contains introduction slides about this workshop and embedded Rust.
- [`host-client`](./host-client/) contains a host client used to communicate with the microbit v2
  via a serial interface.
